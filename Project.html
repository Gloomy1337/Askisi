Ολοκληρωμένα Project:

Η εταιρία K&G έχει ολοκλήρωση μέχρι τώρα διάφορα Project στην γλώσσα προγραμματισμού Java.

Project σε Εξέλιξη:

Τα project με τα οποία ασχολείται η εταιρία μας προς το παρών βρίσκονται εδω.

Μελλοντικά Project:

Η εταιρίας μας σκέφτεται να φτιάξει πλατφόρμα εκπαίδευσης, να δημιουργήσει μελλοντικά Project στις γλώσσες προγραμματισμού Java, C++ και ανάπτυξης ιστοσελίδων με Javascript, CSS, HTML και PHP.
